HTML

I also practiced creating a basic navigation system for my website. I built a simple navigation bar and used anchor tags to link between multiple pages. This helped me refresh my understanding of how websites connect different pages together and how users navigate through a site.

Terminology

  • Heading: A title or subtitle used to organize content on a webpage.
  • Paragraph: A block of text that provides information or content on a webpage.
  • Link: An element that allows users to navigate to another page or section of a webpage.
  • Button: An interactive element that can be clicked to perform an action, such as navigating to another page.
