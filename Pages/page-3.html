Week 4

Flexbox

This week I spent some time working with flexbox/(flexible box) this is a CSS layout module that provides an efficient way to distribute space and align items in a container. even when their size is dynamic. The main reason to use Flexbox is to create responsive and flexible layouts that adapt to different screen sizes and orientations. This works well better than inline or block which work horizontally and vertically.

CSS Layout and Design

I also spent time improving the design of my webpage using CSS. I used div elements to separate sections of the page and applied styles to control spacing, alignment, and visual structure. By experimenting with margins, padding, and positioning, I was able to make the layout clearer and more consistent across the page.

Flexbox terminology

  • Flex Container: The parent element that holds the flex items.
  • Flex Items: The child elements of the flex container.
  • Justify Content: The property that defines how flex items are distributed along the horizontal axis.
  • Align Items: The property that defines how flex items are aligned along the vertical axis.

Week 3

CSS Styling Refresher

During last week I refreshed my understanding of CSS and how it controls the visual appearance of a webpage. I practiced applying colors, fonts, spacing, margins, and padding to HTML elements to make the page look more organized and visually appealing.

CSS terminology

  • margin: Space outside an element.
  • padding: Space inside an element around its content.
  • border: The line around an element.
  • width: Sets the element's width.
  • height: Sets the element's height.
  • color: Text color.
  • font-size: Size of the text.
  • font-family: Font type used for text.
  • text-align: Horizontal alignment of text (left, center, right).
  • text-decoration: Adds underline, line-through, or none to text.
